# Facial-Geometry-Adjustment-Preview
Landmark-based facial adjustment preview tool for visual demonstration purposes. (Chin, Nose, and Nose bridge)



## Features
- Chin size presets (Small / Medium / Large)
- Nose width & tip refinement presets
- Nose bridge smoothing (angle-aware)
- Before / After toggle
- Zoom & vertical pan preview
- Webcam capture or image upload
- Deterministic, repeatable results

## Disclaimer
This tool is provided for demonstration and visualization only.
It does not represent medical advice, cosmetic outcomes, or guaranteed results.
Actual results may vary significantly.

## Tech
- JavaScript
- HTML5 Canvas
- MediaPipe FaceMesh
- CSS
